# 匿名树洞

一个运行在 AstrBot 上的跨微信号匿名群聊插件。用户通过私聊 Bot 加入群组后，发送给 Bot 的消息会以随机 emoji 身份匿名广播给其他成员。

## 功能

- 支持加入、退出、查看成员、查看帮助。
- 支持文字、图片、语音、视频、文件的匿名转发。
- 默认采用虚拟群聊模式，首次私聊普通消息时自动加入并中转。
- 支持在虚拟群聊模式和私聊模式之间切换。
- 行为日志会写入 AstrBot 宿主日志，可直接在 Web 管理面板查看。

## 模式说明

- 虚拟群聊模式：插件接管私聊消息，匿名转发给其他成员，默认不走 LLM。
- 私聊模式：插件不再接管消息，交回 AstrBot 默认流程，可继续走 LLM 或其他插件。

## 指令

- /join：切换到虚拟群聊模式
- /leave：切换到私聊模式
- /members：查看当前成员
- /anon_status：查看当前模式
- /anon_help：查看帮助

## Web 端查看日志

本插件不会单独新增一个页面，而是复用 AstrBot 现有日志页。

1. 启动 AstrBot 并打开 Web 管理面板。
2. 进入日志页面。
3. 搜索 [匿名树洞] 或 action= 关键字，即可筛选本插件行为日志。

日志会记录以下行为：

- 插件加载与卸载
- 用户加入、重复加入、退出、帮助查看、成员查看
- 匿名消息广播成功、部分失败、无人可收的跳过情况
- 通知发送失败、多媒体无法转发等异常

为避免直接暴露原始来源标识，日志中的 sender 和 target 会使用稳定哈希脱敏显示。
